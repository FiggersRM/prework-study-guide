HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Using tags h1 thru h6 creates different headers in a webpage
  • Elements that have other elements inside are known as parent elements, and the elements inside are known as child elements.
  • Semantic elements are important because they help communicate with other developers about information on the webpage, and the browser, as well as a wide variety of other users.
  • Lists can be unordered lists, ordered lists, or list items
  • The img tag is used to convey information about images on a webpage, using the src tag for image location and alt for an alternate description
  • Use the br / tag to create a break between two blocks of content

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want aaround the content inside an element

Git

  • Git is used for developers to code locally while not interrupting other developers potentially working on the same code
  • GitHub is an online repository for code and is very critical to not only editing and pulling code, but also reverting to old code in the case of a bug
  • There are many helpful git commands that can be used in the CLI to help developers:
  • git status: checks what branch we are currerntly working on
  • git checkout -b branch-name: creates a new branch under the new name and switches to it
  • git pull: Pulls the most recent form of code from the online repository
  • git add: Stages a branch of code to prepare it to commit
  • git commit -m: commits the staged code to a pull request to GitHub

Javascript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer exectues code in a script
  • Conditionals are code structures that are used to test if an expression returns true or not
  • Functions are a way of packaging functionality that you wish to reuse by taking arguments
  • Loops and conditional statements help us utilize control flow so that code isn't interrupted
  • Arrays are a group of data stored in a single variable
  • Each part of an array is assigned an index within the defined variable, starting at 0
  • for loops can be used to efficiently work through an array of data
  • Functions are a set of instructions which tell the computer how to perform a certain task
  • functions do not automatically execute when a script is run, but must be defined and called upon
  • A function is defined using a declaration, and executed when the definition is typed and followed by parenthesis
